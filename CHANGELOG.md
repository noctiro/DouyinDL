# Changelog

## [1.2.0] - 2026-05-13

### 新增

1. 多线程下载支持，根据文件大小自动调整线程数以提升速度
2. 支持在下载进行中随时取消任务
3. 下载进度页面新增剩余时间估算显示
4. 视频封面支持点击放大预览，并可单独下载封面图片
5. 支持预测性返回手势
6. 新增多语言支持
7. 支持应用切至后台时继续下载
8. 下载完成后推送系统通知

### 修复

1. 修复下载速率指示器数值闪烁的问题

## [1.1.0] - 2026-05-12

### 新增

1. 下载前的文件大小指示
2. 下载时的下载速率指示
3. 清晰的下载错误信息提示
4. `关于`对话框

### 修复

1. 资源泄露问题
2. 某些情况下视频会打开失败
